# TP ML Semiconductor

Ce projet porte sur l'analyse et la modélisation des données de fabrication de semi-conducteurs (SECOM).

## Structure du projet
- `data/`: Contient les datasets (secom.csv).
- `notebooks/`: Notebooks Jupyter pour l'exploration et le prototypage.
- `src/`: Scripts Python pour le prétraitement, l'entraînement et l'évaluation.
- `results/`: Dossier pour stocker les modèles sauvegardés et les rapports.
- `docs/`: Documentation supplémentaire.
